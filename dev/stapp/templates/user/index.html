Community Management Hub

This project is made by Mohamed Bilel Besbes. It is a tool to upload large data volumes through excel and csv, and to filter them through the UI to download them as csv or excel. This project is basically a sample project to make a DevOps project, therefore the development part is not that important.

What you can do

Upload

Upload your big volumes of data in csv or excel format

Filter

Filter the data through the UI without trouble

Download

Download the data in excel format for your convenience. Excel is the number one platform when it comes to analyzing data, finding information, preparing charts, and presenting them to decision-makers. Excel performs well for all levels of user

Deeper look

Conveniently import and export your data
